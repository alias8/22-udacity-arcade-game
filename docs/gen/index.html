Front-End Web Developer Nanodegree

Assignment submission for Udacity Front-End Web Developer Nanodegree "Project: Classic Arcade Game Clone". Play from the GitHub Pages or use a web server to run index.html and play my version of frogger from the source. This is a fork of the original here. Features improvements to original with collision detection with svg path instead of bounding rectangle.

Install

No dependencies required

Game Rules

  • Use the keyboard arrow keys to try to reach the water and collect gems
  • Avoid the falling rocks and bugs

Built With

  • Paper.js
  • JSDoc 3
